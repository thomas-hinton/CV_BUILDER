# CV Builder

Application web multi-utilisateurs pour créer et publier un CV en ligne (profil, formations, expériences, compétences), avec un backend **FastAPI** et une base **SQLite**.

## Fonctionnalités de la release V1.0.0

La version complète du projet est sur la branche **`V1.0.0`** :

- **Authentification** — inscription, connexion, JWT
- **CV structuré** — profil, formations, expériences, compétences
- **Publication** — CV public accessible par lien (`/cv/{slug}`)

## État des branches

| Branche   | Statut |
|-----------|--------|
| `main`    | Branche d'accueil — présentation du projet et point d'entrée du dépôt. |
| `V1.0.0`  | **Release V1.0.0** — code source, documentation d'installation, tests et historique de développement. |

Pour installer et lancer l'application :

```bash
git checkout V1.0.0
```

Puis suivre les instructions d'installation dans le README de cette branche.
